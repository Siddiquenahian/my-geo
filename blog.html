What is event bubble?

Oct 18, 2022

Event Bubble..

Event Bubbling is a process in the DOM where we can passed all the events through the element class order. When an element is generate by an event, it will pass the event up through its parent and rare elements in the DOM tree until it gets to its higher level element.

What is a callback function and why will you use it?

Dec 01, 2022

Call back function...

A callback is a function that is passed as an argument to another function, which is then executed by the outer function to perform a specific task. Callback functions are necessary because many JavaScript tasks are not carried out in a linear fashion. Instead, it runs in the background while the main code is still running. The purpose of a callback is to run code in reply to a situation.
